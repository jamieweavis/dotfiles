# dotfiles

> 👨‍💻️ My macOS dotfiles - configuration for zsh, aliases, brew, casks, git, vim, iTerm & Raycast

## Installation

Sync dotfiles to home directory

```bash
./bootstrap.sh
```

Install brew dependencies & casks

```bash
brew bundle
```
